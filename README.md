# Student Management System (Java)

A console-based Student Management System built using Java.

## Features
- Add Student
- View Students
- Search Student
- Update Student
- Delete Student
- Save Data to File
- Load Data from File

## Concepts Used
- OOP
- ArrayList
- File Handling
- Exception Handling
- Loops & Conditions
- CRUD Operations

## How to Run

```bash
javac Project.java
java Project
